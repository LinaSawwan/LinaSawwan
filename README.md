## Churn Prediction Model 

**Description:**  
The aim of this project is a predictive model using Orange Telecom‚Äôs dataset to identify customers likely to churn.

- Conducted data exploration, feature engineering, and preprocessing, including handling class
imbalance.
- Evaluated multiple models (Logistic Regression, Random Forest, XGBoost, SVC), selecting and
fine-tuning Random Forest for optimal performance.
- Achieved 84% recall and 81% F1-score, prioritizing recall to better detect at-risk customers.
- Identified key churn factors and provided data-driven retention strategies and business insights.
